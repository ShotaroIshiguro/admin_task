Q：2014年時点でのブランド展開と現状を元に、東急ホテルズの将来戦略は何か、「東急REIブランド」への統合は正しい決断だったかを分析せよ

将来戦略
（調査内容）
https://www.tokyuhotels.co.jp/company/message/index.html 
	” 私たちはホテルを単なる宿泊・食事・集いの場にとどめず、「まちや地域を共にデザインし、旅やイベントのデスティネーションとなる基点」として育てていきたい”
	” 世界の旅の目的地となっている「SHIBUYA」をはじめとした大都市から、自然豊かなリゾートまで、日本国内の魅力あふれる立地に、幅広いブランドのホテルを運営しています”
https://www.tokyu.co.jp/history/chapter09_6_1/ 
	コロナ禍以降、運営受託（MC）方式を出店の主軸とし、環境変動に左右されにくい柔軟な契約形態へ移行。老朽化等の課題を抱える店舗は賃貸借契約満了をもって撤退し、店舗の新陳代謝を加速させる方針を明確化した。
	二子玉川エクセルホテル東急・渋谷ストリームエクセルホテル東急等は、東急グループの不動産開発案件と連動して開業しており、グループの都市開発プロジェクトにホテルを組み込む形で好立地を確保するモデルが確立されている。
	インバウンド送客の強化に向け、上海・シンガポールを含む海外4拠点にオフィスを開設し、国際的な送客網を整備している。
	需要予測に基づくレベニューマネジメントの強化により収益最大化を図り、2016年度には東急ホテルズ設立以降過去最高となる売上高856億円・営業利益38億円を達成した。
（現在のブランド戦略のまとめ）
	現在の各ブランドは顧客層・価格帯・立地によって明確に役割が分担されており、上質を志向するすべての顧客層をもれなくカバーすることを目的としたブランド再編が行われている。
	カプセルホテルやドーミインのような低価格帯ホテルは運営しておらず、東急ブランドの安心感・信頼性を維持することが戦略の大前提となっている。
（将来戦略のまとめ）
1.	高単価帯の顧客層へのアプローチ強化。インバウンドを中心とした上質志向の旅行者ニーズに対応するブランド展開を推進し、価格競争に依存しない収益構造を目指す。
2.	東急グループの都市開発と連動した出店モデルの継続・拡大。インバウンド需要の集中する大都市・主要観光地において、グループ開発案件へのホテル組み込みを通じて好立地を効率的に確保する。
3.	ホテル資産の所有・賃借から、運営受託（MC）方式への移行を推進。固定費を抑制したスリムな経営基盤を構築することで、需要変動リスクへの耐性を高める。

（東急ホテルズ将来戦略に関するロジックツリー）
収益最大化
	売上げ向上
		東急グループ都市開発と連動した出店計画
		旅やイベントのデスティネーション化による集客
	収益率向上
		インバウンド需要の取り込み
		ブランド再編による高価格帯顧客層への集中
		需要予測に基づくレベニューマネジメントの強化
リスク最小化
	運営受託方式への転換（資産保有リスクの低減）
	一部部門の外部委託化（固定費削減）
	積極的な新陳代謝（赤字店舗の撤退）

東急REIブランドへの統合の評価
ブランド統合の目的として以下が考えられる。
1.	極端な低価格路線との差別化による利益率維持
2.	顧客層のニーズを網羅的にカバーしながらブランド数削減による管理費削減
以上の目的の達成度は以下の通りであり、ブランド統合は正しかったといえる。
1.	統合後も全ブランドで上質志向のポジションを堅持し、他ホテルとの価格競争に引き込まれることなく差別化を維持した。統合翌年の2016年度には東急ホテルズ設立以降の過去最高業績を記録したことは、統合の有効性を裏付けている。
2.	5ブランドを3ブランドに集約したことで顧客・社内の両面でブランド体系が簡素化され、会員基盤の一本化も実現した。想定した上質志向の顧客層の網羅という点でも目的は達成されている。
