# 損益計算書

- 損益計算書の仕組み
    - 売上総利益（粗利）= 売り上げ高 - 売上原価
    - 営業利益 = 売上総利益(粗利) - 販売費及び一般管理費
    - 経常利益 = 営業利益 + 営業外収益 + 営業外費用
    - 税金等調整前当期純利益 = 経常利益 + 特別利益 - 特別損失
    - 当期純利益 = 是均等調整前当期純利益 - 住民税・法人税等

- 経常利益とは、本業での利益である営業利益から財務活動などによる収益・費用（営業外）を加味したものであり、税引前当期純利益は、経常利益に一時的な損益である特別利益・特別損失を加味した数字。

# 収益と費用

### 収益の認識
1. 現金主義: 顧客から入金された時に認識
2. 実現主義: 経済的価値の増加が確定（商品の発送など）した時に認識
- 実現主義が期間損益認識の原則

### 費用の認識
1. 現金主義: 商品やサービスの対価を支払った時に認識
2. 発生主義: 実質的に商品を使ったりサービスを受けたりした時に認識
- 発生主義が期間損益認識の原則
- 発生主義において、期をまたいで価値が生じる費用を支払ったときは、その期に相当する部分のみを費用と認識

### 費用収益対応の原則
- 今期に発生した費用の内、今期の収益獲得に貢献した部分だけを今期の費用とする
- 今期の利益に貢献しなかった費用分は、棚卸し資産として賃借対照表に計上

# 賃借対照表

- 賃借対照表とは、事業を行うために、どのようにお金を使ったか、そのためのお金をどのように集めたのかをまとめたもの
- 資産: お金の使い方
- 負債・純資産: お金の集め方（返す必要があるかどうか）
- 資産 = 負債 + 純資産

### 資産
- 流動資産: 一年以内に現金化する予定がある（現預金・売掛金・棚卸資産など）
- 固定資産: 建物・土地・有価証券など

### 負債
- 流動負債: 一年以内に支払が必要（買掛金・短期借入金など）
- 固定負債: 社債・長期借入金

### 純資産
- 自己資本・株主資本
- 損益計算書の当期純利益は利益剰余金として純資産に組み込まれる


# キャッシュフロー計算書

- キャッシュフロー計算書とは、会社の現金の流れを表したもの
- 営業CF: 本業からどれだけの現金を得られたか
- 投資CF: 将来のためにどれだけ投資しているか
- 財務CF: どれだけお金を借りて、返済したか

### 営業CF
- 商品・サービスを提供することで得た現金と、材料費や人件費などで支払った現金の差
- 営業CFがプラスとは、事業から儲けが上がっている

### 投資CF
- 有形固定資産や有価証券の売却時と購入時の現金の差
- 投資CFがプラスとは、資産を売って現金を得ている
- 投資CFがマイナスとは、資産を購入している（将来に投資している）ことを指し、成長時の企業にありがち

### 財務CF
- 財務CFがプラスとは、借入などにより現金が入っている
- 財務CFがマイナスとは、借入金の返済などにより現金が出ていく
- 株式の発行や配当金支払いも含まれる

# 財務3表のつながり
- 損益計算書(P/L)、賃借対照表(B/S)、CF計算書
1. P/Lの当期純利益 = B/Sの純資産の繰越利益剰余金
2. B/Sの資産 = B/Sの負債と純資産
3. B/Sの資産の現預金 = CF計算書の現金及び現金同等物の期末残高
