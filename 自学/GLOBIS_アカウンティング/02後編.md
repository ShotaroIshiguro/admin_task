# アカウンティング・事業活動

# 事業活動の定量化と財務三表

- 損益計算書 P/L: 儲かり具合（売上 - コスト = 利益）
- 賃借対照表 B/S: お金の調達の使い道（資産 = 負債 + 純資産）
- CF計算書: お金の流れ

# 損益計算書 P/L

### P/Lの読み方
- 売上高: 事業の規模感をつかむ
- 営業利益: 本業でどれだけ儲けているか
- 経常利益: 持続して利益を出せているか
- 当期純利益: 最終的に当期の利益はいくらか

### P/Lの原則
- 費用は発生主義、収益は実現主義により、それぞれの原因または活動が行われたタイミングで行う
- 収益を計上するときは、その売り上げを得るために使った費用のみを同じ期のP/Lに計上する（費用収益対応の原則）

# 賃借対照表 B/S

- 倒産の恐れは、流動資産と流動負債の差額を見るとわかる

### 減価償却
- 固定資産の取得に要した支出を使用期間（耐用年数）にわたって費用計上すること（B/Sの資産からP/Lの償却費(P/L)に移動）
- 定額法: 毎期一定金額を償却する
- 定率法: 固定資産の耐用期間中、毎期の期首の資産価値に一定の割合（償却率）を乗じた金額を償却する

# キャッシュフロー計算書

### 営業CFの表示
- 直接法: 経理担当者が直接数字を打ち込む
- 間接法: P/LとB/Sの数字から算出

# 運転資本

- 原材料の買掛金の受け取りと商品の売掛金の支払いにはタイムラグがあり、その期間で足りなくなる資金が運転資本
- 運転資本 = 売上債権 + 棚卸資産 - 仕入責務
- 売掛金を早く受け取る（売上債権を小さく）・在庫を圧縮（棚卸資産を小さく）・買掛金を遅く支払う（仕入責務を大きく）すると運転資本は減ることで経営しやすくなる
